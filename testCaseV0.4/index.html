Key, Value

garnet은 기본적으로 key, value 시스템으로 운영됩니다. 다중 파라미터 전달 시 ( key, value ) 쌍으로 연결이 되며 key만 입력 시 value 값이 리턴됩니다. 그 외에는 Dom 객체가 리턴됩니다. 
				// dk.Dom().S( $k, $v );

				var dom = dk.Dom().S( 'width', '100%' );
				dom.S( 'height', 100 );
				dom.S( 'position', 'absolute' );

				dom.S( 'left', 100, 'top', 100, 'display', 'none' );

				var domWidth = dom.S( 'width' );

dk.Dom

Element 객체 생성 - dk.Dom 객체

dk.Dom 메서드는 selector 또는 element 생성을 통해 dk Dom 객체를 반환하는 메서드 입니다.
이 메서드는 Indexed collections 을 내장한 객체를 반환합니다. 내부 프로퍼티로 domList, dom, elList, el, length를 가지고 있습니다.
domList : Dom 리스트, dom : 첫번째 Dom, elList : native element 리스트, el : 첫번째 element

기본으로 div element가 생성됩니다. 
					dk.Dom();

					/* log
					DomList {domList: Array[1], elList: Array[1], length: 1, dom: Dom, el: div}
					domList:Array[1]
					dom:Dom
					elList:Array[1]
					el:div
					length:1
					*/

S method

key, value 시스템

key, value 시스템이 적용됩니다.
기본적으로 해당 엘리먼트의 스타일 객체에 값이 입력됩니다. '@' 가 붙은 문자열로 입력 시 엘리먼트의 attribute 객체에 입력됩니다.
수치는 px 없이 숫자값으로 입력 가능합니다. 퍼센트는 '10%', 문자열로 입력하면 됩니다. 리턴값은 px 값들은 숫자로 그외는 문자열로 리턴됩니다. 
					// dk.Dom().S( $k, $v );
					var dom = dk.Dom().S( 'left', 100, 'width', 100, 'height', 100 );
					dom.S( 'top', '100%' );
					dom.S( '@id', 'test' );
					dom.S( 'left' ); // 100
					dom.S( 'top' ); // '100%'
					dom.S( '@id' ); // 'test'

css vendor prefix

vendor prefix는 자동으로 입력됩니다. 
					// box-shadow, -ms-box-shadow, -moz-box-shadow, -o-box-shadow, -webkit-box-shadow
					dk.Dom().S( 'boxShadow', '0px 0px 10px #62615f inset' );

UTIL

dk.random

$max와 $min 사이의 랜덤 숫자를 반환합니다
앞의 숫자가 큰 숫자 입니다. $min 생략시 0 ~ $max 사이의 램덤 숫자를, 둘 다 생략 시 0 ~ 1 사이의 랜덤 숫자를 반환합니다. 
					dk.random(); // 0 ~ 1
					dk.random( $max ); // 0 ~ $max
					dk.random( $max, $min ); // $min ~ $max

dk.randomInt

$max와 $min 사이의 랜덤 정수를 반환합니다
앞의 숫자가 큰 숫자 입니다. $min 생략시 0 ~ $max 사이의 램덤 숫자를 반환합니다. 둘다 생략은 불가능 합니다. 
					dk.randomInt( $max ); // 0 ~ $max
					dk.randomInt( $max, $min ); // $min ~ $max

dk.Sheet - SPRITE SHEET

sprite sheet는 여러 개의 작은 그래픽을 바둑판식 격자 배열로 포함하고 있는 비트맵 이미지 파일입니다.
여러 그래픽을 단일 파일로 사용함으로써 Animate 및 모바일 버튼 등의 활용에서 한 파일만 로드하여 그래픽을 사용할 수 있도록 합니다.

sprite sheet 객체 생성 - dk.Sheet 객체

dk.Sheet 메서드는 sprite sheet 이미지와 데이터 파일을 로드하여 sprite sheet 객체를 반환하는 메서드 입니다.
dk.Dom을 확장하여 구현되어 있습니다. 
				dk.Sheet( $cb, $img, $json/*, $framerate*/ );

dk.Sheet().S()

dk.Dom을 확장하여 사용하기 때문에 dk.Dom의 모든 매서드를 사용할 수 있습니다. 
					dk.Sheet( function(){}, 'asset/box.png', 'asset/box.json' ).S( 'position', 'absolute', '<', 'body' );

dk.Sheet().S2()

dk.Sheet의 개별 매서드들은 S2 매서드를 통해 사용할 수 있습니다. 
					var sheet = dk.Sheet( function( $sheet ){}, 'asset/box.png', 'asset/box.json' );

					// 반복 재생
					sheet.S2( 'repeat' );

					// 1회 재생
					sheet.S2( 'play' );

					// 정지
					sheet.S2( 'stop' );

					// 뒤로 재생
					sheet.S2( 'rewind' );

					// 해당 프레임 이동 후 정지
					sheet.S2( 'gotoAndStop', frame );

					// 해당 프레임 이동 후 1회 재생
					sheet.S2( 'gotoAndPlay', frame );

					// 구간 반복
					sheet.S2( 'abRepeat', frame1, frame2 );
